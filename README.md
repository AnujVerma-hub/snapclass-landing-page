# SnapClass Landing Page

A modern and responsive landing page built for showcasing the **SnapClass AI Attendance System**.  
Designed to present the product vision, features, development journey, and technology stack with a clean UI and smooth user experience.

---

## 🌐 Live Demo

🔗 Live Website: https://snapclass-landing-page-six.vercel.app/

🔗 GitHub Repository: https://github.com/AnujVerma-hub/snapclass-landing-page

---

## ✨ Features

- Responsive modern UI
- Clean navigation system
- Hero section with product introduction
- Interactive features showcase
- Development journey timeline
- Tech stack presentation
- Optimized deployment setup
- Lightweight and fast-loading design
- Mobile-friendly layout

---

## 🧩 Website Sections

### 🔹 Navigation Bar
Provides smooth navigation across all sections of the website.

### 🔹 Hero Section
Introduces SnapClass and highlights the core vision of the AI-powered attendance system.

### 🔹 Features Section
Showcases the main functionalities and capabilities of the platform.

### 🔹 Journey Section
Displays the development journey and evolution of the project.

### 🔹 Tech Stack Section
Highlights the technologies used in the development of the project.

### 🔹 Footer Section
Contains project links, copyright information, and additional navigation.
